# Epic: resume

Owns official resume and cover-letter materials for software engineering job applications.

## Goal

Maintain a durable master inventory of experience, projects, accomplishments, skills, formatting goals, and tailoring rules so Maverick can generate a polished resume or cover letter for a target job without starting from scratch.

## Scope

- Master experience inventory.
- Resume formatting and style rules.
- Tailored resume variants.
- Cover-letter drafts.
- Job-specific application notes.

## Out Of Scope

- Portfolio website implementation except when sharing career facts.
- Broad visual site polish.
- Inflating or inventing experience.

## Rules

- Default target is software engineering.
- Tailoring should select from durable facts, not fabricate claims.
- Cover letters should be concise, specific, and grounded in the role and inventory.
- Generated outputs should be saved as inspectable artifacts in this repo.
